### How to use the theme

Go to VSC and do the `Ctrl + K`, `Ctrl + T` combinations, and type the name of the theme you want. Here you have the name's of the themes:

- 4am session *(blue)*
- 4am session *(aqua)*
- 4am session *(green)*
- 4am session *(red)*
- 4am session *(orange)*
- 4am session *(yellow)*
